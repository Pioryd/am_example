# Artificial Mind - App Editor

## Install and run

- install MongoDB
- install [AM Framework](https://github.com/Pioryd/am_framework)
- configure config.json
- ```powershell
  npm link am_framework
  npm install
  npm start
  ```

## Used managers of [AM Framework](https://github.com/Pioryd/am_framework)

- core_admin_server
- core_editor
- core_backup
- core_admin_scripts
